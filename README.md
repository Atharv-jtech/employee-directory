# Employee Directory

A MERN Stack Employee Directory using login feature application developed during internship training.

## Features

- Add Employees
- View Employees
- Activate / Deactivate Employee
- Delete Inactive Employees
- MongoDB Database Integration
- REST API using Express.js
- React Frontend

## Tech Stack

### Frontend

- React
- Axios
- CSS

### Backend

- Node.js
- Express.js

### Database

- MongoDB

## Project Structure

```
client/
server/
```

## Installation

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
